# Proyecto de Automatizacion & QA - Matías Emanuel Barreiro

## Descripción

Este es un proyecto de automatizacion de pruebas realizado en lenguaje Python,
usando frameworks como Pytest y Selenium Webdriver.

El objetivo del proyecto consiste de automatizar distintas pruebas funcionales
de una aplicación web, un E-commerce llamado "Swag Labs"

## Tecnologías usadas en el proyecto
- Python
- Selenium Webdriver
- Pytest
- Pytest HTML
- Git
